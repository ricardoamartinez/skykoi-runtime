# `SKYKOI tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
SKYKOI tui
SKYKOI tui --url ws://127.0.0.1:18789 --token <token>
SKYKOI tui --session main --deliver
```
